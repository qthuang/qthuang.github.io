Android Virtual Avatar & Calorie Calculator

Introduction

This is an App that combines the function of a calorie counter and an interactive avatar. Android Studio & JAVA were used to develop this app.

How to Run

Import the folder into Android Studio and run the app on an Android Device Emulator, or connect an actual device to run.

How to Use

The user can edit the weight data according to their own data, and the app will calculate their desired daily calorie input. By clicking on the "feed" button, the user can input wanted calories (the use's actual input for each meal) for the pet and the process bar of hunger will change accordingly. The image of the avatar will change if the input exceeds the desired quantity, so that the users will notice that they should not eat more on that day. At the beginning of a day, click on "reset" button to reset the hunger progress bar.
By clicking on the avatar, it responses with simple animation and conversation bubbles.
