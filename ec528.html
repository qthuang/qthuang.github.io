EBPF Networking Library

Introduction

The goal of this project is use OPENED to extract modules from various networking programs, then identify common modules used accross these networking and place these common modules into a library which can be used by many of emerging engineering groups/startups as they develop new networking programs.

  • Identify most commonly used functions across Katran and Cilium that should be part of a network library. Identification can start by doing a simple comparison of the function names/signatures extracted by OPENED from the different libraries.
  • Use OPENED to extract required functionality from the projects.
  • Use OPENEDâ€™s Hook Point transformation functionality to make library usable across hook points.

    • OPENED

    OPENED is joint work with RedHat/IBM to analyze eBPF programs and decompose them into small modules tha tcan be used to create arbitrary programs. OPENED was recently accepted and presented by the Linux Plumbers Conference. There is ongoing work to make OPENED a set of production ready DevTools.
