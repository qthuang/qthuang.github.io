Predicting Medical Expense

Introduction

This is the final project of CS542 Machine Learning class. We used a dataset on Kaggle called "Medical Cost Personal Datasets" to train ML models which predicted personal medical expenses. We used different ML methods includin Linear Regression, Random Forest, Classification and Clustering to train different method and tested their perfomances. The process included data pruning, data visualization, training, hyperparameter tuning, and performance evaluation.
